## üìö Experiment Syllabus (10 Labs)

1.  **Data Visualization:** Analysis of distributions and outlier detection.

9.  **Probabilistic Learning:** Naive Bayesian Classifier for Facial Recognition.
10. **Unsupervised Learning:** k-Means Clustering for medical data analysis.
